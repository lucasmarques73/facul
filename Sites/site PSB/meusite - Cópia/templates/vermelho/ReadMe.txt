*** Instalando template no Joomla
---------------------------------------

Para instalar um modelo exportado e compactado através do painel de administração do Joomla faça o seguinte:
1. Ir para Joomla Administrator (www.your-site.com/administrator) -> Extensões -> Instalar / Desinstalar
2. No "Extension Manager" escolher a primeira opção "Upload Package File".
3. Clique no botão "Browse ..." botão para selecionar o arquivo zip do seu computador.
4. Clique no botão "Upload File & Install" botão.

Para mais informações visite http://docs.joomla.org/How_to_install_templates

*** Utilizing Menus
-------------------------------
Por favor, utilize os seguintes passos para utilizar o estilo de menu concebido com Artisteer:
1. Ir para Joomla Administrator (www.your-site.com/administrator) -> Extensões -> Module Manager.
2. Abrir um menu existente ou criar um novo e colocá-lo no "user3 posição".

NOTA: A "posição user3" pode conter apenas um único menu, ou nenhum.

*** Personalizando o rodapé
------------------------------
Para personalizar o modelo de rodapé no Joomla um lugar de administração ou vários módulos em
"copyright" de posição. Isso irá substituir o texto de direitos autorais padrão incluídos no modelo
rodapé com o novo conteúdo fornecido pelo módulo ..

Aqui estão os passos de exemplo para configurar o rodapé personalizado:
1. Ir para o Administrador do Joomla (www.your-site.com/administrator) -> Extensões -> Module Manager.
2. Clique em "Novo", selecione "Custom HTML", selecione "Next".
3. Nas propriedades do módulo especificar:
    Title - Show Title: No
    Position: copyright
    Custom Output: (o conteúdo do rodapé desejado)
4. Salve as alterações.

Agora você pode utilizar o módulo recém-criado para personalizações rodapé ainda sem a utilização de módulos adicionais.
